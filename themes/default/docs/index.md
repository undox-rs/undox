# Default Theme

The default undox theme provides a clean, responsive documentation layout with light and dark mode support, full-text search via Pagefind, and helpful template features.

## Features

- **Light/Dark Mode** - Automatic theme switching based on system preference, with manual toggle
- **Full-Text Search** - Powered by Pagefind, accessible via `/` keyboard shortcut
- **Responsive Layout** - Sidebar navigation collapses on mobile
- **Syntax Highlighting** - Code blocks with language-specific highlighting
- **Table of Contents** - Auto-generated from page headings
